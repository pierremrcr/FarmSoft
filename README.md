# FarmSoft

Création d'un programme permettant de gérer les stocks de matières premières, dépenses et ressources d'un agriculteur.

Le programme permet de :

1/ Rentrer une nouvelle activité (nourrir ses animaux, vendre ses produits etc.)
2/ Consulter sa banque
3/ Consulter ses stocks

Prérequis :

Vous devez disposer de java sur votre machine. Pour plus d'information : https://www.java.com/fr/download/

Pour tester le jeu :

1/ Télécharger le fichier farmSoft.jar
2/ Créer un dossier et y glisser ce fichier
3/ Depuis le terminal de votre PC ou mac, placez vous dans le dossier où se trouvent le jar
4/ Exécutez la commande : java -jar farmSoft.jar 
5/ Le programme se lance
